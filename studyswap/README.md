# StudySwap - 留学生专属二手交易平台

## 🎯 产品愿景

StudySwap 是专为留学生群体打造的安全、便捷的二手物品交易平台。我们理解留学生在异国他乡的特殊需求，致力于为他们提供一个可信赖的社区交易环境。

## ✨ 核心特色

### 🎓 专为留学生设计
- **学生身份验证**: 通过学生证、录取通知书等方式验证身份
- **校园定位**: 基于学校和地区的精准匹配
- **多语言支持**: 中英文界面，适应不同国家学生需求
- **信用体系**: 基于交易记录建立的信任评级

### 📱 智能分类系统
- **教材书籍**: 专业教材、参考书、学习资料
- **电子设备**: 手机、电脑、平板等数码产品
- **家居用品**: 家具、厨具、装饰品
- **交通工具**: 自行车、滑板、电动车
- **服装配饰**: 衣服、鞋子、包包、饰品

### 🔒 安全保障
- **担保交易**: 平台托管确保交易安全
- **实名认证**: 学生身份验证机制
- **评价系统**: 买卖双方相互评价
- **举报机制**: 维护良好交易环境

### 🌟 用户体验
- **响应式设计**: 完美适配手机和电脑
- **智能搜索**: 支持关键词、价格、位置筛选
- **即时消息**: 买卖双方实时沟通
- **季节推荐**: 根据学期时间智能推送

## 🛠️ 技术栈

### 前端
- **Next.js 15**: React全栈框架
- **TypeScript**: 类型安全
- **Tailwind CSS**: 原子化CSS框架
- **Lucide React**: 图标库

### 后端
- **Prisma**: 数据库ORM
- **SQLite**: 轻量级数据库
- **NextAuth.js**: 用户认证

### 部署
- **Vercel**: 前端部署
- **Railway**: 数据库托管

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd studyswap
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置环境变量**
   ```bash
   cp .env.example .env
   # 编辑 .env 文件设置数据库连接等配置
   ```

4. **初始化数据库**
   ```bash
   npx prisma migrate dev
   npm run seed
   ```

5. **启动开发服务器**
   ```bash
   npm run dev
   ```

6. **访问应用**
   打开浏览器访问 `http://localhost:3000`
